# Expense Tracker

**Автор:** Красняков Артем Андреевич
**Дата:** 04.05.2026

## Описание
Приложение с графическим интерфейсом для учёта личных расходов.  
Позволяет добавлять траты, фильтровать по категории и дате,  
автоматически считать сумму за выбранный период,  
сохранять данные в JSON и загружать их при запуске.

## Как использовать
1. Установите Python 3.6+.
2. Запустите `expense_tracker.py`.
3. Для добавления расхода заполните поля:
   - Сумма (положительное число)
   - Категория из выпадающего списка
   - Дата в формате `ГГГГ-ММ-ДД`
4. Нажмите **«Добавить расход»**.
5. Для фильтрации выберите категорию и/или укажите диапазон дат, нажмите **«Применить фильтр»**.
6. Сумма отфильтрованных записей отображается внизу.

## Примеры использования (тесты)

### Тест 1: Добавление корректной записи
- Вход: сумма=150.50, категория=Еда, дата=2025-01-10
- Результат: запись добавлена в таблицу, сохранена в `expenses.json`

### Тест 2: Ошибка при вводе отрицательной суммы
- Вход: сумма=-50
- Результат: всплывает сообщение "*Сумма должна быть положительным числом*"

### Тест 3: Фильтрация по дате
- Данные: 2025-01-01 (100р), 2025-01-05 (200р), 2025-01-10 (300р)
- Фильтр: от 2025-01-02 до 2025-01-09
- Результат: показана только запись на 200р, сумма за период = 200
